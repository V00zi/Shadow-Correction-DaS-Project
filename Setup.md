-----------------------------------DEPENDENCIES-----------------------------------------





Install the following python packages with the given pip commands



Pandas

pip install pandas



Pytrends

pip install pytrends



GTab

pip install gtab



-------------------------------------STRUCTURE-------------------------------------------



The repo contains 3 main folders each necessary to be able to reproduce the study locally please keep the names of the folders unchanged 



* Data-sets: contains all the extracted datasets for the study
* Scripts: contains the necessary python files to run the tests on the data
* Results: contains the result generated during our study (Final report based on these results)





----------------------------------------SETUP---------------------------------------------



1. Read this file (Du bist smart :O)
2. Clone the repo or download the whole project as a ZIP (some Data-set files may take up more storage)
3. Install all the dependencies mentioned above using the pip command
4. Proceed to "Scripts" folder (This will be your root folder to run the tests)
5. Follow the instructions in "scripts-readme.txt" to run the tests for yourself



P.S if you just want to see the produced results instead go to the "Results" folder and follow the instructions in "results-readme.txt"
